Worked as UX Engineer Intern in summer 2016 with Google Identity Team. Here showcases my engineering project - Goolge Account UI Front-end Development.

GOOGLE ACCOUNT UI IMPLEMENTATION

In 2015, Google launched a new identity with stronger emphasis on clean, minimal layouts for frictionless and extensible design. As a result, Google Account sign-in and sign-up flow on all devices and platforms need to be updated. My project is to implement the new design on iOS and macOS.

OBJECTIVES & CHALLENGES

USER EXPERIENCE AT HEART

As Google’s products touches millions of users everyday and Google Accounts is highly visible on every product, the impact of this project is considerable to both direct consumers and the company as a whole. While developing, we focused on the following aspects.

User Trust

Implementing new UI across all the platform promotes consistency while enhancing user trust of the new UI while promoting brand recognition.

Accessibility

Great product should be accessible to everyone. I learned the technical practices that provide all users with enjoyable experience.

Functionality

User experience is both about how it looks and how it works. Maintainable and scalable code base is the foundation for a quality product.

PROCESS & DELIVERABLES

COLLABORATION WITH TEAMS

Implementing projects entails a great deal of team effort. I have frequently attended meetings with PMs, designers, copywriters and engineers to help determine the timeline and level of effort for each task before initiation of a specific task. I also provide detailed documentation to track and explain my process. Languages and technologies I used to deliver my work include Soy (HTML), GSS (CSS), JavaScript.

RESULT & IMPACT

PROJECT LAUNCHING

The project launched in November 2016 impacting 2 million users. The code I delivered met of Google’s coding standard and requirements from user experience team. completed supporting tasks for successful launch, including: webdriver tests and test plans, accessibility review, design review and launch plan.
